# 第5章 Kintoneユーザー管理

事務局から依頼を受けた際のKintoneユーザーの追加・無効化手順です。

---

## 5-1. ユーザー追加の依頼を受けたとき

事務局から新入居者のKintoneユーザー登録依頼を受けたら、以下の情報を確認します。

- 追加する人物の氏名
- ログインに使用するメールアドレス（またはログインID）

---

## 5-2. Kintoneユーザーの追加手順

1. Kintoneの **管理画面**（サイボウズ共通管理）にログインします
   - URL：`https://（ドメイン）.cybozu.com/admin/`
   - Kintone管理者権限が必要です

2. **「ユーザー管理」** → **「ユーザーの追加」** を選択します

3. 以下を入力して保存します

   | 項目 | 内容 |
   |------|------|
   | ログイン名 | メールアドレスまたは任意のID |
   | 表示名 | 氏名 |
   | パスワード | 初期パスワードを設定（本人に伝える） |

4. 追加後、**ユーザーのログイン名（コード）** を確認しておきます
   - 事務局が M_組合員 への登録時に使用します

5. 事務局にユーザー追加が完了したことを通知します
   - 通知する情報：ログイン名（Kintoneユーザーコード）

> **Kintoneへのアクセス権限について**
> ユーザーが該当のKintoneアプリ（M_組合員など）を閲覧・操作できるよう、必要に応じてアクセス権限を設定してください。権限設定はKintoneの「アプリの設定 → アクセス権」から行います。

---

## 5-3. Kintoneユーザーの無効化手順

退去等で事務局からユーザーの無効化依頼を受けた場合の手順です。

1. Kintoneの **管理画面** にログインします
2. **「ユーザー管理」** から対象ユーザーを検索します
3. ユーザーの **「有効/無効」** を無効に変更して保存します

> **削除ではなく無効化を推奨します。**
> 削除するとKintone上のレコード（M_組合員や請求レコード等）との紐付けが失われる可能性があります。
